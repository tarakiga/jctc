# Create a New Case

Learn how to create and configure a new investigation case in JCTC.

---

## Prerequisites

- You must be logged in with **Investigator** or **Admin** role
- You need basic case information ready

---

## Step-by-Step Guide

### Step 1: Navigate to Cases

1. Click **Cases** in the side menu
2. Click the **+ New Case** button

![New Case Button](../assets/screenshots/new-case-button.png)

---

### Step 2: Fill in Basic Information

Complete the required fields:

| Field | Description | Required |
|-------|-------------|----------|
| **Case Title** | Descriptive name for the case | âœ“ |
| **Case Type** | Select from dropdown (e.g., Cybercrime, Fraud) | âœ“ |
| **Priority** | 1 (Critical) to 5 (Low) | âœ“ |
| **Status** | Usually starts as "Open" | âœ“ |
| **Description** | Detailed case summary | Optional |

!!! warning "Case Numbering"
    The Case Number is auto-generated by the system. Do not try to manually set it.

---

### Step 3: Set Case Scope

Choose the scope of your case:

- **Local**: Investigation within Nigeria
- **International**: Cross-border investigation requiring MLAT

---

### Step 4: Add Initial Details

Fill in additional information:

1. **Intake Channel**: How the case was received (Walk-in, Hotline, Email, etc.)
2. **Reporter Type**: Who reported (Victim, NGO, Law Enforcement, etc.)
3. **Risk Flags**: Tag urgent cases (Child Safety, Imminent Harm, etc.)

---

### Step 5: Save the Case

1. Review all entered information
2. Click **Create Case**
3. You'll be redirected to the case detail page

!!! success "Case Created!"
    Your case is now visible in the Cases list and can be assigned to team members.

---
