# Keyboard Shortcuts

Master these shortcuts to navigate JCTC like a pro.

---

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| ++ctrl+k++ | Open global search |
| ++escape++ | Close modal / Cancel action |
| ++question++ | Show keyboard shortcuts help |

---

## Navigation

| Shortcut | Action |
|----------|--------|
| ++g++ then ++d++ | Go to Dashboard |
| ++g++ then ++c++ | Go to Cases |
| ++g++ then ++e++ | Go to Evidence |
| ++g++ then ++i++ | Go to Intelligence |
| ++g++ then ++r++ | Go to Reports |
| ++g++ then ++s++ | Go to Settings |

!!! tip "How to use"
    Press ++g++ first, release, then press the second key within 1 second.

---

## Case Management

| Shortcut | Action |
|----------|--------|
| ++n++ | New case (when on Cases page) |
| ++e++ | Edit current case |
| ++ctrl+s++ | Save changes |

---

## Evidence

| Shortcut | Action |
|----------|--------|
