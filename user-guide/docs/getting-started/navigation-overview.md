# Navigation Overview

Learn how to navigate the JCTC interface efficiently.

---

## Main Layout

The JCTC interface consists of four main areas:

```
┌─────────────────────────────────────────────────────┐
│  Top Navigation Bar                                  │
├──────────┬──────────────────────────────────────────┤
│          │                                           │
│  Side    │                                           │
│  Menu    │           Main Content Area               │
│          │                                           │
│          │                                           │
└──────────┴──────────────────────────────────────────┘
```

---

## Top Navigation Bar

| Element | Description |
|---------|-------------|
| **Logo** | Click to return to Dashboard |
| **Search** | Global search across all modules |
| **Notifications** | View alerts and updates |
| **Profile Menu** | Account settings, logout |

---

## Side Menu

The side menu provides access to all modules:

| Icon | Module | Description |
|------|--------|-------------|
| :material-view-dashboard: | **Dashboard** | Overview and quick stats |
| :material-folder: | **Cases** | Case management |
| :material-devices: | **Evidence** | Device and evidence tracking |
| :material-lightbulb: | **Intelligence** | Intelligence reports |
| :material-shield-check: | **Compliance** | NDPA compliance |
| :material-file-chart: | **Reports** | Analytics and reports |
| :material-cog: | **Admin** | System administration |

!!! tip "Collapsing the Menu"
    Click the **hamburger icon** (☰) to collapse the side menu for more workspace.

---

## Keyboard Shortcuts

Press `?` anywhere to view available keyboard shortcuts.

| Shortcut | Action |
|----------|--------|
| `Ctrl + K` | Open search |
| `G` then `D` | Go to Dashboard |
| `G` then `C` | Go to Cases |
| `G` then `E` | Go to Evidence |
| `?` | Show shortcuts help |
